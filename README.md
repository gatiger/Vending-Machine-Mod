# Vending Machine Mod

A NeoForge vending machine mod for Minecraft 1.21.1.

This mod adds player-owned vending machines that can sell items for configurable currency. Machines support 9 sale slots, owner/admin protection, cashbox payments, saved inventory when moved, custom display signs, and visible sale items behind glass.

## Current Status

Beta: `0.1.0-beta`

This is an early test build. Please test in a backup world or a new world.

## Features

- 2-block-tall vending machine
- 9 configurable sale slots
- Player-owned machines
- Admin/config GUI
- Cashbox for collected payments
- Machine keeps settings and inventory when moved
- Configurable display signs
- Sale items visibly render inside the machine
- Crafting recipes and JEI support

## Download

Go to the **Releases** page and download the latest `.jar`.

Do not download the source code ZIP unless you want to build the mod yourself.

## Installation

1. Install Minecraft 1.21.1 with NeoForge.
2. Download the latest mod jar from Releases.
3. Put the jar in your instance's `mods` folder.
4. Launch the game.
